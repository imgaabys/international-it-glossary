# 🌍 Glossário de TI: Brasil 🇧🇷 vs Portugal 🇵🇹 vs Inglês 🇺🇸

## 💡 Notas Culturais para Suporte em Portugal
* Em Portugal, é muito comum utilizar o termo **"Informática"** para se referir ao departamento de TI de forma geral.
* O atendimento ao utilizador tende a ser ligeiramente mais formal (uso do "você" ou "o senhor/a senhora") do que o padrão brasileiro.
* O uso de termos em inglês é extremamente comum em empresas de tecnologia (como *Daily*, *Sprint*, *Backlog*), tal como no Brasil.
